# 🌐 https://deva-2408.github.io/

Personal portfolio website built as a single HTML file.

## About

A dark, minimal portfolio showcasing my projects, skills, and background.
Built from scratch with HTML, CSS, and vanilla JS — no frameworks, no dependencies.

## Features

- Single self-contained `.html` file (no build step, no npm)
- Fully responsive (mobile + desktop)
- Smooth scroll-reveal animations
- Profile photo embedded as base64
- Direct links to live projects

## Projects Showcased

- **Ledora** — Independent digital publication → [ledoraog.lovable.app](https://ledoraog.lovable.app/)
- **Repkit** — Workout tracker → [repkit.lovable.app](https://repkit.lovable.app/)

## Deploy

Works on any static host. Recommended:

- **GitHub Pages** — push to a repo, enable Pages in Settings → Pages → main branch
- **Netlify** — drag and drop the `.html` file at [netlify.com/drop](https://app.netlify.com/drop)
